# 7-Minute-Workout-App
This app provides a 7-minute workout to the user. It also has the feature to calculate BMI for their personal growth. It will store the history of the user in database which helps user to check his daily performance. It also has a sound effect alert on every exercise so that users can be aware. The user will get a finished screen after completion of the workout.
