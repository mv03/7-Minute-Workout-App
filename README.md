# 7-Minute-Workout-App
This app provides 7 minute workout to the user.It also has the feature to calculate BMI for thier personal growth.It will store the history of the user in database which helps user to check his daily performance.It also has the sound effect alert on evry exercise so that user can be aware.The user will get a finish screen after completion of workout.
